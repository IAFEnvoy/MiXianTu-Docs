# 人间界简介
《尚书》记载，九州为：冀州、兖州、青州、徐州、扬州、荆州、豫州、梁州、雍州。人间界设定也为此九州。

人间界是五大场景中第二小的地图，XY坐标范围均为-600~600，九州地图如下：（有点难看，将就一下吧）

```
    -600 -200    200   600
 600 +-----+------+-----+
     | 兖州 | 青州 | 徐州 |
 200 +-----+------+-----+
     | 扬州 | 冀州 | 荆州 |
-200 +-----+------+-----+
     | 豫州 | 梁州 | 雍州 |
-600 +-----+------+-----+
```
